# 🎙️ Dubbing Artist Backend API

A scalable and secure backend for a **Dubbing Artist Platform**, built using Node.js and Express.  
This API handles user authentication, profile management, and media uploads for dubbing artists.

---

## 🚀 Tech Stack

- **Node.js** – Runtime environment
- **Express.js** – Backend framework
- **MongoDB** – NoSQL database
- **Mongoose** – ODM for MongoDB
- **JWT Authentication** – Secure user authentication
- **OAuth 2.0 (Google Auth)** – Social login integration
- **Cloudinary** – Cloud storage for images and videos

---

## ✨ Features

- 🔐 User authentication (JWT-based)
- 🔑 Google OAuth 2.0 login
- 👤 User profile creation & management
- 🎙️ Dubbing artist profile (languages, skills, etc.)
- 📁 Media upload (audio/video/images) via Cloudinary
- ⚡ Scalable and modular backend architecture
- 🛡️ Secure environment variable handling
